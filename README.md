> ⚠ **重要声明：本项目仅用于技术研究、自动化测试、自有网站的数据统计验证。

---

## 📌 项目简介

珊瑚PV 是一个基于 Python + Selenium 的 **网站访问行为自动化测试工具**。  
它可以帮助开发者在测试环境中模拟真实用户访问，用于：

- 测试网站统计系统（PV/UV/VV 等）是否正常记录  
- 测试网站的跳转逻辑、点击链路是否有效  
- 检查网站性能、页面加载速度  
- 在开发阶段快速生成测试流量数据  
- 自动化替代人工反复访问、点击

所有访问行为仅用于 **本地测试数据** 生成，不能用于任何商业或非法目的。

---

## ✨ 功能特点

- 🔍 **模拟真实用户访问信息（PV / UV / VV）**
- 🖱️ **可自动点击站内链接、跳转页面**
- 📊 **可视化界面运行 / 后台隐形运行（无头模式）**
- ⏱️ **可自定义访问时间、停留时长、访客数量**
- 🌐 **支持自定义 User-Agent、代理等配置**
- 🧪 **适用于测试统计脚本、行为分析、流程验证等技术用途**
